### v2.1 修正：波长带改绝对像素单位（2026-06-10 实测反馈后）

首版 v2 把谐波带定义为"每周长周期数"，实测暴露两个问题：小岛的细节带波长被压缩成
~4px 锯齿、大陆只剩长波（**跨形状的周长依赖**——形状内的边长依赖修掉了，但换了个尺度回来）；
三带 RMS 归一 + 权重混合 + 整体峰值归一把 B/C 带挤成 A 带上的毛刺（**三层不可见**）。修正：

- **波长/振幅全部改为绝对像素**：A 带（宏观位移）360~900px/±28px、B 带（波动）90~220px/±9px、
  C 带（细节）22~55px/±3px；每个形状按自身周长换算整数谐波，容不下宏观波长的小岛自动跳过该带，
  并带平滑淡入（P 达到波长下限 1.5 倍时满幅）避免相邻尺寸外观突变。
- **每带独立峰值归一后按绝对振幅叠加**，三层互不挤压；小图形另有 0.045/0.018/0.008×周长护栏。
- **角点帐篷上限**：位移 ≤ 离角点弧距 × 0.8·tan(内角/2)（凹角再乘 0.45）——绝对振幅变大后，
  仅靠衰减窗口不足以阻止尖角两侧沿不同法线位移后折叠（实测尖三角曾因此连环自交坍缩到回退分支）。
- 采样步长 = 最小波长 / 4（自动满足 Nyquist），质量档位只控制采样点数上限（256~2048）。
- 实测位移统计：周长 2700/1000/760 的三个形状 mean 均为 10~12px、max ~30px（质感一致）；
  周长 360/140 的小岛按护栏降为 mean 2.6/1.3px，无锯齿。

## 0. 该不该做（前提判断）

**该做，且只能重构、不能调参。** 理由：用户观察到的两个症状（包络抹杀噪声、效果随原始边长漂移）不是参数没调好，而是同一个结构性根因的两个表现——**噪声场以"边"为单位参数化**（`t ∈ [0,1]` 逐边、相位逐边随机）。这个设计下：

- 包络 `sin(πt)^1.15` 是**结构必需品**：噪声在每条边的 `t=0/1` 处相位不连续，去掉包络就会在原始顶点处产生跳变和折角。包络不是可以删掉的装饰，它是给逐边噪声"擦屁股"的。
- 所以"把包络调弱/指数调小"救不了：包络弱了顶点处接缝就露出来；包络强了噪声就被压扁。这是一对死锁，调参只能在两个坏结果之间滑动。

**最强反对意见**：MVP 阶段是否值得为视觉效果投入一次算法重写？这是价值判断，不在本报告管辖范围。本报告只论证：(a) 现状缺陷是结构性的；(b) 重构成本可控（核心改动集中在 `naturalize_vertices` 一个函数 + 一个新噪声模块，验证/回退管线全部保留）。

**可证伪条件**：如果实际产品中用户绘制的草稿顶点非常密（顶点间距已接近期望的海岸线特征波长，比如自由手绘连续采样），那么"逐边"和"全局"参数化的差异会自然消失，本报告主张的重构收益就不成立。从现有测试样例（4～8 个稀疏顶点）和 MVP 文档看，草稿是稀疏多边形，前提成立。

## 1. 现状算法流程（速览）

```text
原始多边形（n 个顶点）
  └─ 逐边处理（for edge_index in 0..n）：
       ├─ 细分数 = f(边长, 周长占比, 归一化边长)        ← 边长决定
       ├─ 振幅   = clamp(边长 × base × √归一化, 2, 0.025×画布)  ← 边长决定
       ├─ 对每个细分点 t：
       │    offset = 振幅 × sin(πt)^1.15 × 三层正弦噪声(edge_index, t)
       │    offset 再被安全限幅（0.35×边长、0.75×角距、凹角/尖角系数、
       │                         最近非邻边距离×0.38、关键点距离×0.5）
       │    沿"该边固定法线"位移
  └─ 去重 → Laplacian 平滑(2 遍, w=0.16) → 自交/关键点校验
  └─ 失败则按 0.75/0.55/0.40 降扰重试 → 再失败回退纯平滑 → 再失败返回原轮廓
```

## 2. 根因分析

### 2.1 包络为什么"抹杀"了三层噪声

`envelope = sin(πt)^1.15` 在每条边两端强制归零，量化看：

| 位置（占边长） | 包络保留的振幅 |
|---|---|
| 距顶点 5% | ≈ 12% |
| 距顶点 10% | ≈ 26% |
| 距顶点 20% | ≈ 54% |

即**每条边有 40% 的区段（两端各 20%）噪声振幅被压到 54% 以下**，顶点本身被钉死为零位移。后果有三层：

1. **包络自己成了最显眼的"形状"。** A 层噪声频率为每边 1.0～4.5 个周期，和包络（半个周期的正弦鼓包）处于同一尺度。两者相乘后，视觉上占主导的是"每条边一个对称鼓包"，三层噪声退化为鼓包表面的小涟漪——这正是"被包络抹杀"的数学解释。
2. **所有原始顶点是位移零点 → 扇贝状外观。** 轮廓呈现"一边一瓣"的花瓣/扇贝模式，周期性与原始顶点严格对齐，人眼立刻识别出底层多边形。
3. **噪声特征永远无法跨越顶点。** 噪声场被原始顶点切割成 n 个互不连通的"格子"，任何海湾/半岛特征都被困在单条边内，不可能出现横跨两条边的大尺度地貌。

### 2.2 "效果因原始边长不同而不同"的四个来源

1. **空间频率 ∝ 边长。** 波频单位是"每边周期数"：同样 2 个周期，400px 的边产生波长 200px 的起伏，40px 的边产生波长 20px 的起伏。同一形状内长短边的海岸"质感"（粗糙度的空间尺度）完全不同。
2. **振幅与限幅 ∝ 边长。** 名义振幅 = `边长 × 1.0 × √归一化`，再被 `0.025×画布` 截断。实算下来：画布 1000×640 时上限 16px，**任何边长 > ~16px 的边名义振幅都顶满上限**，看似恒定；但随后 `constrain_coastline_offset` 的 `0.35×边长` 和三角帐篷 `0.75×边长×min(t,1-t)` 对短边（< ~45px）全程生效——短边实际是被"边长比例限幅"统治的，长边是被"画布上限"统治的。两种统治方式不同 → 长短边行为分裂。
3. **细分预算 ∝ 边长。** 段数公式含 `归一化边长` 与 `周长占比` 两个边长项。
4. **位移方向逐边固定。** 整条边的所有细分点沿同一条法线位移，相邻边法线方向跳变，靠包络归零来掩盖折痕——又回到 2.1 的死锁。

### 2.3 附带缺陷（重构时应一并处理）

- **C 层噪声欠采样（接近甚至低于 Nyquist）。** C 层最高 11.6 周期/边，而典型细分只有 17～32 段，即每周期仅 1.5～4.9 个采样点。每周期 < 2 点时纯粹是混叠伪影，根本画不出第三层细节。
- **Laplacian 平滑随后把残存高频磨光。** w=0.16 两遍后，每周期 3 点的成分只剩 ~27%，每周期 2.2 点的只剩 ~14%。**"三层噪声"实际只有 A 层和部分 B 层活到了输出**，C 层先混叠再被磨平。
- **Laplacian 平滑有收缩效应**，轮廓整体向内缩，加剧"被限制在原始边内"的观感。
- **安全降扰是全局连坐。** 一处自交触发整条轮廓 0.75/0.55/0.40 的全局降幅，问题点和无辜区段一起变平。

## 3. 候选方案对比

| | 方案 A：周长域噪声（推荐） | 方案 B：中点位移分形细分 | 方案 C：2D 域噪声场 |
|---|---|---|---|
| 思路 | 全轮廓按弧长 s∈[0,P) 统一参数化，构造周期连续的多倍频噪声 f(s)，沿平滑法线位移 | 递归细分每条边，中点沿法线随机位移，振幅随深度衰减 | 在画布平面上定义 fBm 噪声场 f(x,y)，轮廓点按所在位置取值位移 |
| 消除顶点钉死 | ✅ 噪声连续跨越顶点，无需包络 | ❌ 原顶点仍是不动点 | ✅ |
| 消除边长依赖 | ✅ 频率/振幅以世界单位（周长比例/像素）定义 | ⚠️ 初始振幅仍逐边，需额外归一 | ✅ |
| 闭合无缝 | ✅ 整数谐波天然周期闭合 | ✅ | ✅ |
| 新增依赖 | 无（沿用现有 hash 确定性随机） | 无 | 需 noise crate 或自写 2D value noise |
| 与现有管线兼容 | 高：constrain/校验/降扰/回退全保留 | 中：递归结构与现有逐点限幅不合 | 中 |
| 风险 | 顶点不再精确保留（见 §6 风险） | 改善有限，2.1 问题只解决一半 | 法向位移仍需弧长重采样，等于 A + 额外依赖 |

**推荐方案 A**。理由：同时根治两个症状；不引入依赖；保留 `shape.id+name` 哈希确定性（同一形状每次生成结果一致，符合 MVP 文档的确定性预期）；改动半径最小。方案 C 本质是 A 的 2D 版本，多付一个依赖却没多换来什么；方案 B 保留顶点钉死，只值得作为"必须精确过原始顶点"需求出现时的备选。

## 4. 推荐方案 A 详细设计

### 4.1 管线

```text
原始多边形
  ① 等弧长重采样：步长 ds = clamp(P / target_points, ds_min, ds_max)
     —— 采样密度与原始顶点分布完全解耦（顶点细分不变性的基础）
  ② 周期噪声场 f(s)：三个倍频带的整数谐波叠加（见 4.2）
  ③ 局部衰减 g(s)：仅在凹角/尖角的弧长窗口内衰减（替代全边包络，见 4.3）
  ④ 平滑法线 n(s)：中心差分切线旋转 90°（替代逐边固定法线）
  ⑤ 位移：p(s) = q(s) + n(s) × clamp(A × g(s) × f(s), ±max_offset(s))
     —— max_offset 沿用现有安全限幅，但基准从"边长"换成"局部特征尺寸"（见 4.4）
  ⑥ Taubin 平滑（λ/μ 双步，无收缩）替代 Laplacian
  ⑦ 校验/降扰重试/回退管线 —— 原样保留
```

### 4.2 周期噪声场（无依赖、确定性、天然闭合）

对闭合轮廓，最简单且严格无缝的构造是**整数谐波谱合成**：

```text
f(s) = Σ_octave w_o · f_o(s)
f_o(s) = Σ_{k ∈ K_o} a_k · sin(2πk·s/P + φ_k)
```

- `k` 取整数 → `f(0) = f(P)` 严格成立，闭合无缝，不需要任何接缝处理。
- 三个倍频带以**整周周期数**定义（替代现在的"每边周期数"）：
  - A 带（宏观海湾/半岛）：k ∈ [2, 6]
  - B 带（中尺度）：k ∈ [9, 18]
  - C 带（细碎纹理）：k ∈ [28, 56]
- `φ_k`、`a_k` 由现有 `hash_unit(seed ^ salt_o ^ k)` 派生，确定性保持；`a_k ∝ k^(-β/2)`（β ≈ 1.6~2.0，分形海岸线的典型谱斜率），每个带做 RMS 归一后按 `w_o` 加权。
- 谐波数 ~30 个，每采样点 O(30) 次 sin，512 个采样点共 ~1.5 万次三角函数，微秒级，无性能问题。

**采样密度与 C 带匹配**：`target_points` 取 ≥ 4 × max(K_C)（如 k_max=56 → ≥ 224 点，默认 256～512），保证最高频带每周期 ≥ 4 个采样点，根治 2.3 的欠采样。

### 4.3 角点衰减（替代全边包络）

包络原本要解决的真问题只有一个：凹角/窄缝处大位移容易自交。新方案改为**外科手术式**处理：

- 对每个原始顶点计算现有的 `vertex_corner_offset_factor`（尖角/凹角判定逻辑保留）。
- 仅在该顶点两侧弧长 `r_c` 窗口内（`r_c ≈ 1.5 × P/k_min(C带)`，并以相邻边长一半为上限）用 smoothstep 把振幅从 `corner_factor` 过渡回 1.0。
- 凸尖角可以**不衰减或轻微衰减**——尖角外凸形成海岬反而自然；凹角维持现有 `0.45` 强衰减。

效果：衰减只发生在真正危险的角点邻域（占周长的小比例），其余 80%+ 轮廓上三层噪声全振幅工作。

### 4.4 安全限幅基准从"边长"换成"局部特征尺寸"

`constrain_coastline_offset` 的框架保留，但：

- 删除 `0.35×边长`、`0.75×角距三角帐篷`（它们是边长依赖的主要来源，且与新的角点窗口衰减重复）。
- 保留并升级：`最近非邻边距离 × 0.38`（防窄缝自交，这是真正干活的约束）、`关键点距离 × 0.5`（防关键点出界）、全局振幅上限 `A`。
- "非邻边"判定从"边索引相邻"改为"弧长距离 < 阈值"（重采样后没有原始边索引概念）。
- 复杂度仍是 O(n·m)（n 原始边数、m 采样点数），与现状同阶；编辑器尺度无压力，将来可加网格加速。

### 4.5 全局振幅

```text
A = clamp(amplitude_perimeter_ratio × P, amplitude_min, amplitude_canvas_ratio_max × 画布短边)
```

单一全局值，整条轮廓质感一致。`amplitude_perimeter_ratio` 默认 ≈ 0.02（可从现状的视觉效果反推校准）。

### 4.6 平滑

Taubin 平滑（λ=0.33, μ=-0.34，1~2 遍）替代 Laplacian：去高频毛刺但不收缩。由于 4.2 已保证采样密度，平滑需求本身也大幅下降，甚至可以默认 0 遍、仅在降扰重试分支启用。

`CoastlineParams` 经 `request.meta.ext.coastlineParams` 由前端下发（`WorldMapPanel.tsx` 按质量档位传参），属于 API 模型改动，按 AGENTS.md 要求附兼容说明：

- 前端 `WorldMapPanel.tsx` 的四档质量参数需同步改为新字段（档位差异主要落在 `targetPoints` 与各带谐波上限）。
- 后端对旧字段**容忍但忽略**（serde 字段都是 `Option`，天然向后兼容），过渡期前后端可不同步发布。

## 6. 风险与未决问题

1. **原始顶点不再被精确保留。** 现状算法顶点钉死（位移为 0），新算法顶点同样会被噪声位移。如果产品语义是"用户画的点必须出现在最终轮廓上"，方案 A 构成回归——**需要产品确认**。若需要折中，可对原始顶点加小半径的弧长窗口衰减（同 4.3 机制），把位移压到 ~30% 而非归零。
2. **视觉回归无自动判据。** "更自然"无法单测断言，需人工对照新旧渲染图验收（建议截取等边方形、长短边悬殊多边形、窄湾形状三个用例）。
3. **极小/极细形状**（周长 < ~100px 或宽度 < 振幅量级）下整数谐波带可能拥挤，需要按周长缩减 C 带上限（`k_max ≤ P / (4·ds_min)`），实现时处理。

## 8. 结论

| 症状 | 根因 | 对策 |
|---|---|---|
| 包络抹杀三层噪声 | 逐边噪声参数化迫使包络存在；包络与 A 层同尺度，自己成了主形状 | 全周长弧长参数化 + 整数谐波周期噪声，包络废除，角点改窗口衰减 |
| 效果随原始边长漂移 | 频率/振幅/细分/限幅四项全部以边长为基准 | 全部换成世界单位（周长比例、像素、局部特征尺寸） |
| （附带）第三层噪声不可见 | 欠采样 + Laplacian 磨平 | 采样密度对齐最高频带 + Taubin 无收缩平滑 |

**什么证据会推翻本方案**：(a) 产品要求原始顶点精确保留（→ 改用 §6.1 折中或方案 B）；(b) 实际草稿顶点密集到顶点间距 ≈ 特征波长（→ 重构收益消失）；(c) "顶点细分不变性"测试在新实现上仍不通过（→ 说明边长依赖另有未识别来源，需回到分析）。

0总览：一条流水线

核心目标：把用户随手画的几个点（一个粗多边形）变成一条看起来自然、整体仍贴合草稿、且同一形状每次生成结果一致的海岸线。 整条管线是单向的，只有最后的"校验"环节可能触发回头重试：

图 0：v2 管线总览。①→⑦ 单向；⑧ 校验不通过才回头降扰重试。
三个贯穿全程的设计原则： ① 弧长参数化 —— 噪声沿"整条轮廓的弧长"定义，而非逐条边，从根上消除"长边比短边平滑"与扇贝感； ② 分层与分尺度 —— 宏观/中/细三层独立，宏观随图形缩放、细节用绝对像素； ③ 确定性 —— 所有随机数由 shape.id + name 哈希派生，同一形状永远生成同一结果。

1输入：稀疏草稿

输入是用户在编辑器里点出的一个多边形 MapShapeDraft，通常只有 4 ~ 数十个顶点。 它定义了"这块陆地大致长什么样"，但边是笔直的、角是尖的——不像真实海岸。算法的任务是在不破坏整体形状的前提下，给它注入自然的起伏。

图 1：本文统一使用的示例大陆草稿（6 个顶点）。后续每一步都基于它。
顶点数 < 3 直接原样返回。下面记原始顶点数为 V，是个小常数。

2等弧长重采样

第一步把稀疏多边形沿周长均匀切成 N 个密集的等距点。这是整个算法的地基，原因有二：

  • 解耦原始顶点：之后噪声只认"我在轮廓上走了多远（弧长 s）"，完全不知道原始边/顶点在哪。这样在一条边里插入共线点不会改变结果（顶点细分不变性），也就不会出现 v1 那种"每条边一个鼓包"的扇贝纹。
  • 统一采样密度：步长 step = 最细波长 / 4（默认 18/4 = 4.5px），保证最高频细节每周期至少 4 个采样点，杜绝混叠锯齿。
step = clamp(C带最小波长 / 4, 1, 8) // 默认 4.5px N = clamp(ceil(周长 P / step), 32, max_points) // max_points: 默认 2048 / 印刷 10240
图 2：草稿（红虚线）被重采样成 N 个等弧长点（蓝点）。点沿周长等距，与原始顶点无关。
N 是全程的主导变量，随周长线性增长直到 max_points 封顶。后面所有复杂度都以它为单位。 时间 O(N+V)空间 O(N)

3角点圆化

在加噪声之前，先把手绘的尖角磨圆。理由：自然地形没有精确的尖角；而且若保留尖角，后面噪声会在尖点两侧沿突变的法线方向位移，形成"针刺"伪影。

做法是对重采样后的点跑若干轮 Laplacian 扩散（每点向左右邻居中点移动一半）。妙处在于：直线段上的点本就等于邻居中点、位置不变，只有有曲率的地方（原始角点）才被磨圆。圆化范围 ≈ 步长 × √轮数，轮数由半径推出（默认半径 22px、约 24 轮）。

图 3：左 = 重采样后（尖角）；右 = 圆化后（角变圆，直边纹丝不动）。
时间 O(R·N)，R≈24空间 O(N) R 是圆化轮数，封顶 96。

4三层噪声谱（核心）

这是算法的心脏。海岸线的"自然感"来自不同尺度的起伏叠加：大尺度的海湾半岛、中尺度的波动、小尺度的细碎纹理。我们用三个独立的"噪声带"对应这三层，每一层都是一组整数谐波正弦波的叠加。

为什么用整数谐波？

沿弧长 s∈[0,P) 构造 f(s) = Σ aₖ·sin(2π·k·s/P + φₖ)，其中 k 取整数。整数频率保证 f(0) = f(P)——也就是绕轮廓一圈天然首尾相接，闭合无缝，不需要任何接缝处理。

三个带的分尺度策略

这是 v2 反复迭代后的关键设计——三层用不同的尺度规则：

带 角色 波长 峰值振幅 尺度策略
A 宏观海湾/半岛 P/7 ~ P/2.5 min(0.03·P, 55px) 相对周长：大陆海湾大、小岛海湾小（分形自相似）
B 中尺度波动 P/18~P/7.5，夹在 [40,110]~[90,240]px min(0.018·P, 28px) 半相对：跟随大小但被绝对窗口夹住
C 细碎纹理 18 ~ 45px（绝对） min(0.024·P, 8px) 绝对像素：长直边与小岛每像素粗糙度一致
为什么 C 必须绝对？若细节波长随周长变长，大陆的长直边就会显得比小岛平滑——这正是中途踩过的坑。固定 18~45px 后，全图细节质感统一。

谐波的随机幅度与谱斜率

每条谐波 k 的振幅 = 随机量 × k^(-β/2)，β=1.8（分形海岸线典型谱斜率，高频按幂律衰减）；随机量在 [0.25, 1] 区间取，让带内谱线强弱不均、纹理更不规则。相位 φₖ 也是随机的。这些随机数都由种子哈希确定性派生。最后每带做 RMS 归一再按需缩放。

图 4：三个噪声带各自的波形（横轴 = 沿轮廓的弧长，纵轴 = 归一化位移）。A 长波大摆幅、B 中等、C 细密。它们将叠加成总位移。
性能护栏：波长越细、周长越大，整数谐波数越多（大陆 C 带可达数百条）。若全量求和，成本 = O(N × 谐波数) 会爆炸。所以每带谐波数封顶 96，超出时分层抽样（每层 hash 抖动取一条，保持谱形与确定性）。记每带谐波上限为 H=96。
时间 O(N·H·B)，B=3 带空间 O(N·B + H)

5粗糙度包络调制

如果三层噪声在整条海岸上力度均匀，看起来会像"机器纹"——到处一样的抖动。真实海岸是间歇的：有的岸段礁石嶙峋，有的岸段平缓如滩。

为此再生成一条独立的低频包络（波长 P/8~P/2，归一到 [0,1]），用它去调制 B、C 两带的振幅：包络高处粗糙、低处平静。调制深度 C 带 0.7、B 带 0.35、A 带不调制（宏观形态不该忽强忽弱）。

图 5：上 = 低频粗糙度包络（紫，0~1）；下 = C 带被它调制前后对比——包络低谷处细节被压平，形成"粗糙段 / 平静段"交替。
时间 O(N·7)空间 O(N) 包络只有 ≤7 条谐波，成本可忽略。

6逐点位移与压限

现在遍历 N 个采样点，把噪声变成实际的几何位移。每个点做下面这串运算——这是把"自然感"和"贴合草稿/不自交"两个矛盾目标调和的地方。

6.1 位移方向 = 平滑法线

位移沿轮廓外法线方向。法线由该点左右邻居连线（中心差分切线）旋转 90° 得到——用邻居而非原始边，法线平滑连续，不会在原始顶点处突跳。

6.2 结构层 / 细节层分离

把三带分成两组：结构层 = A + B（决定整体轮廓走向），细节层 = C（海岸的细碎质感）。它们被区别对待，这是控制"贴合度"的关键。

6.3 D_max 软封顶（压限）

问题：A、B 两层在某些点会同相叠加，把轮廓推得离草稿很远（曾达 ±90px），导致"认不出原形状"。 对策：只对结构层（A+B）合计做软封顶到 D_max（默认 35px），细节层 C 始终全振幅叠加、不受封顶。

"软"封顶用 tanh：位移在 D_max 的 60% 以内原样通过，超过后平滑渐近逼近 D_max、永不触顶。这区别于硬截断——硬截断会让一段岸线全部贴在 D_max 处形成"平直墙"；软封顶 + 始终活着的 C 细节，让封顶区仍有起伏，不出平直边。

structural = A + B structural = softcap(structural, min(D_max, 0.3·特征尺寸)) // 软封顶 detail = C // 不封顶 offset = (structural + detail) × 角点衰减

6.4 局部特征尺寸限幅

每个点还会测局部肢体宽度（到对面轮廓的最近距离）。宏观带在细窄肢体上自动熄火（避免把细臂扭断），细节带几乎不受影响。这样细臂紧贴草稿、宽阔腹地照常起大湾——限幅是局部的，不是全局一刀切。

6.5 末端安全限幅

最后再用 0.38 × 特征尺寸（防自交）和关键点距离（防地点出界）做一道软饱和限幅。它与 D_max 各管各的：D_max 管贴合度，这道管几何安全。

图 6：沿轮廓的位移剖面。灰 = 结构层原始值（含越界尖峰）；蓝 = 软封顶后；橙 = 再叠加细节 C。注意结构尖峰被压在 ±D_max 内，但叠加细节后仍处处起伏。
每点成本：角点衰减 O(凹角数)、特征尺寸 O(V)、三带合成 O(B)、关键点 O(L)。 时间 O(N·(V+L))空间 O(N) L = 关联关键地点数。

7Taubin 平滑

位移后可能残留极高频的毛刺。普通 Laplacian 平滑会让整体收缩（轮廓越磨越小）。Taubin 平滑用一收一胀两步（λ=0.33 收、μ=−0.34 胀）抵消收缩，去毛刺但不缩水。默认只跑 1 轮。

时间 O(passes·N)空间 O(N) 之后还有一次相邻点去重 O(N)。

9复杂度总表

符号：N=重采样点数（主导，数百~上万）、V=原始顶点数（小）、H=每带谐波上限 96、B=带数 3、L=关键地点数（小）、R=圆化轮数（≤96）。

阶段 时间 空间 说明
② 等弧长重采样 O(N+V) O(N) 边指针单调推进
③ 角点圆化 O(R·N) O(N) R≈24，封顶 96
④ 三层噪声合成 O(N·H·B) O(N·B+H) 最大常数项（~290 sin/点），但线性
⑤ 粗糙度包络 O(N·7) O(N) 可忽略
⑥ 逐点位移/限幅 O(N·(V+L)) O(N) 特征尺寸每点扫 V 条边
⑦ Taubin 平滑 + 去重 O(N) O(N) passes 默认 1
⑧ 自交校验 O(N²) O(N) 唯一超线性项 · 瓶颈
⑧ 关键点包含 O(N·L) O(1) L 次射线投射
正常路径（一次成功）： T = O(N² + N·(H + V + L)) 主导项随 N 增大 → O(N²) 最坏路径（连环重试）： T ≈ 5 × 正常路径 （几何安全已就位，极少发生） 空间： S = O(N + V) 输出 O(N)
一句话结论：噪声合成虽然常数大但是线性的（谐波上限已摁住），真正卡 100k 点的是 O(N²) 的自交校验。当前默认 N≤2048 一切毫秒级；要扩到十万级点数，先改校验算法，而不是调任何审美参数。
